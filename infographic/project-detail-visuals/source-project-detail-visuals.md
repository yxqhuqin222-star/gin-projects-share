# Project Detail Visuals Source

目标：把项目详情页中简陋的流程配图升级成适配当前网站风格的信息图。

设计要求：
- 风格需要适配当前网站：黑底、克制、专业、产品化。
- 配图不能只是箭头指向，需要更美观、更像可发布的信息图。
- 保留真实项目语义，不编造项目功能。

项目来源：
- 日追踪看板：Summary 计算、Dashboard、Workbook 导出、播报图。
- 小宇宙播客转文章：episode URL、音频下载、通义听悟转写、transcript、结构化播客笔记。
- 钉钉播报控制台：preview、message schema、send API、DingTalk Bot、trigger time。
- 碎片收集页：Inbox、tag、Local Storage、timeline、backup import/export。
- 飞书聊天记录回放：keyword search、高亮、日期筛选、sender filter、图片预览。
