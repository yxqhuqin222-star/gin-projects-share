# Project Detail Visuals

## Overview
生成 10 张 16:9 项目详情信息图，替换现有 `*-2.png` 和 `*-3.png` 流程配图。

## Learning Objectives
The viewer will understand:
1. 每个项目的核心 workflow。
2. 每个项目的关键模块与输出。
3. 项目详情页的视觉风格保持一致。

## Design Instructions

### Style Preferences
- 黑底、克制、专业、产品化。
- 低饱和技术蓝、绿色、少量 amber accent。
- 与当前 Gin 网站视觉风格一致。

### Layout Preferences
- 16:9 landscape。
- linear-progression + technical-schematic。
- 不是简单箭头，而是系统图、节点、输出摘要、网格背景和技术面板组合。
